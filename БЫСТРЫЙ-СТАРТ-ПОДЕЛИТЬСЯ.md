## 🚀 Как работает:

### 1. Пользователь нажимает кнопку "Поделиться"
(круглая кнопка с иконкой в правом верхнем углу)

### 2. Открывается модальное окно с:
- Аватаром эксперта
- Именем и описанием
- Городом
- Направлениями (топ-5)
- Ссылкой на профиль
- Кнопками действий

### 3. Пользователь выбирает:
- **"Поделиться"** → нативное меню (WhatsApp, Telegram, etc.)
- **"Копировать ссылку"** → только URL в буфер
- **"Копировать всё"** → форматированный текст с информацией
